# 排序练习

## 通用排序算法

|  算法  | 时间复杂度  | 空间复杂度  | 是否稳定 | 优化思路 |
|  ----  | ----  | ----  |----  |----  |
| 冒泡排序  | o(n^2) | o(1) | 是 ||
| 选择排序  | o(n^2) | o(1) | 是 ||
| 插入排序  | o(n^2) | o(1) | 是 |二分查找， 希尔排序|
| 希尔排序  | O(n^(1.3~2) | o(1)  | 否 ||
| 快速排序  | o(n*log(n)) | o(n) | 否  |元数小于10时转换为二分插入|
| 归并排序  | o(n*log(n)) | o(n) | 是 |timSort|
| 堆排序  | o(n*log(n)) | o(1) | 否 ||
